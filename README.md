# Password Manager

## 📝 About the Project
The Password Manager is a beginner-friendly Python application that allows users to securely store and manage account credentials. The application provides a simple menu-driven interface to add, view, search, and delete saved passwords using file handling.

## ✨ Features
- Add account credentials
- View saved passwords
- Search credentials by account name
- Delete saved credentials
- Store passwords using file handling
- Simple menu-driven interface

## 🛠️ Technologies Used
- Python
- File Handling
- Functions
- Loops
- Conditional Statements

## 🎯 Learning Outcomes
This project helped me learn:
- Working with file handling in Python
- Managing data using text files
- Implementing CRUD operations
- Using functions for modular programming
- Developing menu-driven console applications

## 🚀 Future Improvements
